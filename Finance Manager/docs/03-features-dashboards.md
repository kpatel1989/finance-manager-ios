# 03 - Features and Dashboards

## Implemented Tabs
- `Dashboard`: net worth timeline/projection (1Y/3Y/5Y), investment growth, expense trend, goals progress, scenario sandbox, rebalancing drift.
- `Accounts`: add multiple accounts of many types, archive accounts, quick snapshot updates.
- `Planner`: monthly expense totals, goals CRUD, recurring contribution planning.
- `Tools`: tax calculator, debt payoff planner (snowball/avalanche), CSV export/import.
- `Settings`: lock/currency/reminder/tax API settings.

## “Everything Now” Extras
- Scenario sandbox.
- Debt planner.
- Rebalancing drift checks.
- Recurring contribution planning.
- CSV export/import.
- Home-screen quick update action via app icon shortcut.

## UX Priorities
- Manual updates in under a few minutes.
- One-place consolidated financial health view.
- Local-only by default with no institution linking.
